# Django Smart Retail

<b>Lua e-commerce</b> - Plataforma de varejo robusta desenvolvida com <b>django e docker</b>, simulando um marketplace de alta performance. O projeto utiliza <b>PostgreSQL</b> e integra APIs externas para gestão de catálogo, evoluindo para funcionalidades de <b>IA</b> como busca semântica e recomendações personalizadas.

🛠️ <b>Tecnologias Utilizadas</b>
* Back-end: Django (Python)

* Front-end: JS, HTMX & CSS

* Banco de Dados: SQLite, PostgreSQL

* Infraestrutura: Docker & Docker Compose

* Gerenciamento de Tarefas: Django Admin customizado

📦 <b>Como rodar o projeto localmente</b>

> Para rodar este projeto, você precisará ter o Docker e o Docker Compose (Docker Desktop) instalados em sua máquina.

<b>Clone o repositório:</b>

<i>Bash</i>

> git clone https://github.com/allans225/django-smart-retail 

> cd django-smart-retail

<b>Suba os containers:</b>

<i>Bash</i>

> docker-compose up --build

* Acesse a aplicação: O servidor estará disponível em http://127.0.0.1:8000/produtos/all/
